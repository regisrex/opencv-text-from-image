# opencv image text extractor
This script reads text from images and prints them

### Libraries used
- `opencv-python`
- `easyocr`

### How to run

1. Clone repo
```bash
$ git clone <repo_name>
```

2. run main.py script
```bash
$ python3 main.py
```
